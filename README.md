# Client React App scaffolding

This is a scaffolding repo that initialise the client app with:
 - npm 
 - webpack
 - jest
 - basic scripting (build, start, watch mode)

-----

## How to use the App: 

#### Install dependencies

```
npm i
```

#### Run the app in dev mode

```
npm start
```

#### Build the app

```
npm run build
```

#### Run unit tests:

```
npm test:unit 
```
